<analysis>**original_problem_statement:** Faça um site para o Conselho Municipal de Educação de Pereiro - COMEP. O site servirá para monitoramento das escolas da rede municipal. O modelo que deve ser seguido é o site de monitoramento do Conselho Estadual, o https://sisp.cee.ce.gov.br/view/escola/loginEscola.aspx.

O painel das escolas será acessado por 18 escolas. Cada usuário (diretor, secretário) só poderá ver, acessar e editar informações referentes a sua escola. Uma escola não pode ter acesso a informações de outra, esse tipo de acesso é restrito ao pessoal do COMEP, através do painel administrativo.

No painel administrativo, adicione um botão de bloqueio/desbloqueio. Após a escola alimentar todas as informações, o COMEP precisará analisar os documentos e, por isso, o painel da referida escola deverá ficar bloqueado temporariamente, enquanto o COMEP conclui a análise e emite parecer.

**User's preferred language**: Português (pt-BR)

**what currently exists?**
Um sistema full-stack para monitoramento de escolas (COMEP). O backend é em FastAPI com MongoDB e o frontend em React com Shadcn/UI.
Funcionalidades principais:
-   **Autenticação:** Login separado para administradores (COMEP) e usuários de escola (Diretor, Secretário), baseado em JWT.
-   **Isolamento de Dados:** Usuários de escola só podem acessar e modificar dados da sua própria instituição.
-   **Gestão de Dados:** CRUD completo para escolas, docentes e quadro administrativo.
-   **Dashboards:** Painéis para admin e escolas com estatísticas. O painel admin possui gráficos de evolução.
-   **Funcionalidades Admin:**
    -   Geração de relatórios em PDF (lista de todas as escolas, detalhes de uma escola).
    -   Sistema de notificações por email (via Resend) para senhas e lembretes.
    -   Mecanismo para bloquear/desbloquear o acesso de uma escola para análise.
-   **Base de Dados:** O sistema está configurado para ser populado com dados de 18 escolas reais, fornecidos pelo usuário.
